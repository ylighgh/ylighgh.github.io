1.1.3. 常量和变量

常量：在程序运行期间其值不发生变化的量，即具体值。

变量：在程序运行期间其值可以发生变化的量，即用来保存和表示数据的符号

  • 变量需遵循先定义后使用；

  • 变量类型由其变量定义时所决定；

  • 变量值可变；
常量分类

  1. 整型常量

    • 十进制

    • 八进制——以0开头

    • 十六进制——以0x开头

  2. 实型常量——仅十进制表示

    • 小数形式

    • 指数形式：字母e(或E)前必须有数字，后必须为整数，如1e3、1,.8e-3、-123e-6、-.1e-3

  3. 字符常量——单个字符组成，定界符英文单引号''

  4. 字符串常量——单个或多个字符组成，定界符英文双引号""

  5. 符号常量

  • 数据后加u或U——表示 无符号 类型，如535u

  • 数据后加l或L——表示 长 整型，如-1L

  • 默认实型常量为double类型，若有后缀"f"或"F"则为float类型

  • 字符串在存储时会自动在最后加一个标识符'\0'

  • 连续两个英文双引号("")表示 空字符

1.2.1. Bubble Sort

	for(i=sizeof(a)/sizeof((a[0]))-1;i>0;i--){
		for(j=0;j<i;j++){
			if (a[j]<a[j+1])
            {
                swap(&a[j], &a[j+1]);
            }
	}
	}

1.2.2. Select sort

	for(i=0;i<sizeof(a)/sizeof((a[0]))-1;i++){
		location=i;
		for(j=i+1;j<sizeof(a)/sizeof((a[0]));j++){
			if(a[location]<a[j]){
				location=j;
			}
		}
		if(location!=i){
			swap(&a[i],&a[location]);
		}

1.2.3. Insertion sort

	for(i=1;i<sizeof(a)/sizeof((a[0]));i++){
		t=a[i];
		for(j=0;j<i;j++){
			if(a[j]>t){
				break;
			}
		}
		if(j==i) continue;
		for(p=i;p>j;p--)
			a[p]=a[p-1];
			a[p]=t;

	}

1.3. Function

1.3.1. swap

void swap(int *x, int *y)
{
    int t = *x; //t=a
    *x = *y;    //a=b
    *y = t; //a=t
}
